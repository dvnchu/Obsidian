[[web]]

**What is a URL? (Uniform Resource Locator)**
is an instruction on how to access a resource on the internet.

![[Pasted image 20251218184517.png]]

This is a URL with all of its features (it does not use all features in every request)

#### Scheme
This instructs on what protocol to use for accessing the resource (HTTP, HTTPS, FTP...)
### User
Some Services require auth to log in, you can put user and password into the URL to log in.

### Host
The [[DNS|Domain]] name or IP address of the server you wish to access

### Port
The port you are going to connect

### Path
The file name or location of the resource

### Query String
Extra bits of info. for example **blog?id=1**

### Fragment
This is a ref to a location on the actual page requested.
