[[web]]
a web server is a software that listens for incoming connections and then utilistes HTTP to deliver web content to its client. 

ex. Apache, Nginx, IIS, NodeJS

A Web server delivers files from what's called its root directory, which is defined in the software settings.


## Virtual Hosts
Web server can host multiple websites with different domain names; This is made using virtual hosts. The server checks the request headers and matches against its virtual hosts. if it finds a match it shows the website else it shows the default web.

Theres no limit of websites on a server.
