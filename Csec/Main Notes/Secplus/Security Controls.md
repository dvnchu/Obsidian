> [!info] Core Concept
> Prevent security events, minimize the impact, and limit the damage.
## Control categories
*Technical controls
*Controls implemented using systems*
### Managerial controls
*Administrative controls*
### Operational controls
*Using people force*
### Physical controls
*Limit physical access*

## Control types
* **Preventive:** Block access to a resource
* **Deterrent:** Discourage intrusion
* **Detective:**  Identify and log intrusion (may not avoid it)
* **Corrective:** Apply control after an event has been detected
* **Compensating:** You don't have enough resources to solve the problem so you "compensate"
* **Directive:** Direct a subject into security compliance
