[[Linux]]

ls    list files
cd   change directory
cat   concatenate(reads file)
pwd  print working directory

\* works as a wildcard
find finds names and folder
we can do find -name  \*.txt for file types

grep allows us to look for the content inside a file

### Shell operators
& -run commands in the background
&& -Combine multiple commands together
\> -it is a redirector we can take an output from a command and direct it elsewhere
\>>  -same as \> but appends rather tan redirecting so nothing gets overwritten


touch -create file
mdkir -make directory
cp -copy a file or folder
mv -move a file or folder
rm -remove a file or folder
file -determine type of a file
